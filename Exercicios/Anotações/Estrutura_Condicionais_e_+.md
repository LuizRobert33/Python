# Estrutura Concidiconal
## Sintaxe
```python
if condição:
    # código executado se condição for verdadeira
else:
    # código se condição for falsa

# Exemplo:
    idade = 18

if idade >= 18:
    print("Você é maior de idade.")
else:
    print("Você é menor de idade.")
```
